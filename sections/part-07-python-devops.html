7Ô∏è‚É£ Generating YAML with Python

import yaml

deployment = {
  "apiVersion": "apps/v1",
  "kind": "Deployment",
  "metadata": {"name": "app"},
  "spec": {"replicas": 3}
}

print(yaml.dump(deployment))

This is commonly used in:

  • Helm templating alternatives
  • Custom deployment tools

8Ô∏è‚É£ CI/CD Script Example

def deploy():
    subprocess.run(["kubectl", "apply", "-f", "deployment.yaml"])
    subprocess.run(["kubectl", "rollout", "status", "deployment/app"])

deploy()

Python scripts are often executed inside CI/CD pipelines.

9Ô∏è‚É£ Error Handling (Production Grade)

try:
    subprocess.run(["kubectl", "apply", "-f", "app.yaml"], check=True)
except subprocess.CalledProcessError:
    print("Deployment failed")

Proper error handling prevents silent failures.
